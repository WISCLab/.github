# WISCLab @ UW Madison

**Wisconsin Intelligibility, Speech, and Communication Laboratory**

🔗 [kidspeech.wisc.edu](https://kidspeech.wisc.edu/)

WISCLab develops tools and conducts research on children's speech intelligibility, communication development, and within related domains. This GitHub organization hosts the open-source software, data utilities, and internal tools that support that work.

## Getting Started

If you're a new lab member or contributor:

1. **Read the repo-level README** for any project you plan to work on.
2. **Install [shred-guard](https://github.com/WISCLab/shred-guard)** as a pre-commit hook for new repositories that touch research data. PHI leaks are a serious compliance risk, this is not optional (even in private repositories)
3. **Request access** to private repositories through @tjmahr.
4. Follow existing code style and conventions within each repository. When in doubt, open an issue or ask.

## Contributing

We welcome contributions to our public repositories. Please open an issue before submitting large changes so we can discuss the approach. All contributions that touch data pipelines should have shred-guard configured.
